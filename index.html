Help / Controls
  • Arrow tools: Click a player, then another player to draw an arrow.
  • Dotted: Same as arrow but dotted line.
  • Free Draw: Turn on ✍️ then draw on field; release to add arrowhead.
  • Ball: Click ⚽ then click field to place one ball (click ⚽ again for next ball).
  • Select: Click any arrow / free-draw line / ball to highlight.
  • Delete: Click 🗑 or press Delete/Backspace to remove selected.
  • Export PNG: Click 📸.
  • Templates: Save 💾 / Load ⬇️.
  • Frames: +Frame then Play, export Video. Movement uses Trans(s).
